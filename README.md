# Weather Forecast Website

This project is a weather forecasting website developed using Python Flask and OpenWeatherMap API.

## Features
- Search weather by city name
- Displays:
  - Temperature
  - Weather description
  - Humidity
  - Wind speed
- Real-time weather data using API

## Technologies Used
- Python
- Flask
- HTML
- REST API
- OpenWeatherMap API
- Requests Library

## Deployment
Deployed using Render.
